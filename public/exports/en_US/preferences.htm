Suggested Categories:

  • Contexts (C/) Add

    Places/people for which to get emails done. E.g. all emails requiring a phone call might be in C/Call, everything for the office might be C/Office, and all emails for a VIP customer might be in 'C/Bob Smith'.

  • References (R/) Add

    Useful types of information (e.g. R/Invoices, R/Meeting Notes)

  • Users (U/) Add

    Share an inbox with a team? Perhaps for customer support. Delegate emails to a co-worker with user labels (e.g. U/Bob, U/Sue, U/Rita).

  • Time (T/) Add

    Retrieve emails based on how long you think they'll take with time labels (e.g. T/5m T/15m T/1hr).

Manage Status Labels

Special Status Labels

You can create any Status labels you want in Gmail using the Status prefix (S/ by default). However, there are 5 Statuses with special meaning in ActiveInbox (e.g. you are 'waiting on' something), and if you want to change the names of these labels in Gmail, you must tell ActiveInbox the new label name to look for and use, using the textboxes below. (Note that you do not have to use any of these special labels; you can safely delete them from Gmail and ActiveInbox will still work).

  • S/ Next
  • S/ Action
  • S/ Waiting On
  • S/ Finished
  • S/ Old

Want to customize your Status labels? Perhaps you want to include the Gmail Star? Check out Turning Emails into Actions.
