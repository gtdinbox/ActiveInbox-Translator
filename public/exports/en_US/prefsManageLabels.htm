What are Label Types in ActiveInbox?

ActiveInbox groups labels that have a specific meaning (e.g. Status labels, Project labels, etc.), simply by giving them a prefix (e.g. 'P/' for Projects - you can change this).

Suggested Label Types:

  • Add

    Contexts (C/)

    Places/people for which to get emails done. E.g. all emails requiring a phone call might be in C/Call, everything for the office might be C/Office, and all emails for a VIP customer might be in 'C/Bob Smith'.

  • Add

    Users (U/)

    Share an inbox with a team? Perhaps for customer support. Delegate emails to a co-worker with user labels (e.g. U/Bob, U/Sue, U/Rita).

  • Add

    Time (T/)

    Retrieve emails based on how long you think they'll take with time labels (e.g. T/5m T/15m T/1hr).

  • Add

    Announcements (SA/)

    Set up labels for service announcements (e.g. SA/Facebook, SA/Twitter/yourusername, SA/LinkedIn), and then create Gmail filters to move these announcements straight into those labels and bypass your inbox, so you can keep your inbox empty and find announcements when you need them.

Can I undo this?

Yes, just go into Gmail's Settings and rename the labels to remove the label prefix that ActiveInbox adds.

What are Special Status Labels?

You can create any Status labels you want in Gmail using the Status prefix (S/ by default). However, there are 5 Statuses with special meaning in ActiveInbox (e.g. you are 'waiting on' something), and if you want to change the names of these labels in Gmail, you must tell ActiveInbox the new label name to look for and use, using the textboxes here. (Note that you do not have to use any of these special labels; you can safely delete them from Gmail and ActiveInbox will still work).
